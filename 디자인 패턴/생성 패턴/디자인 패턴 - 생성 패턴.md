# 디자인 패턴 - 생성 패턴

> ### 생성 패턴

- __객체 생성에 관련된 패턴__
- 객체의 생성과 조합을 캡슐화해  특정 객체가 생성되거나 변경되어도 프로그램 구조에 영향을 받지 않도록 유연성을 제공한다.
- 생성하는 방법을 __캡슐화__ 하여, 각 클래스들의 어떻게 만들어지고 서로 이어지는지를 숨긴다.
- 의존성을 줄이고 재사용성을 늘리는 것이 생성 패턴의 핵심이라고 생각한다.
